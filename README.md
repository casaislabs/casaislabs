## About

I build **protocol-level primitives** and **on-chain infrastructure**, then ship a **control plane** (UI + APIs) to operate, monitor, and validate the system.

My work focuses on:
- composable smart contracts
- Solana programs in Rust
- invariant-driven design
- account abstraction
- staking & DeFi primitives
- automation and agent-based execution

Open to protocol, infra, and research-driven collaborations.
