## Featured Projects

A curated selection of protocol, infrastructure, and systems-focused work.

### AutonomiX
Autonomous AI agent system with ERC-8004 NFTs, on-chain reputation, and x402 micropayment-gated APIs on Base Sepolia.
- [Repo](https://github.com/casaislabs/AutonomiX) · [Live](https://autonomix.casaislabs.com)

### StableGuard
Gas-optimized stablecoin protocol with oracle safeguards, Dutch auctions, repeg mechanics, and comprehensive testing.
- [Repo](https://github.com/casaislabs/StableGuard)

### StakeFlow
Solana staking dApp built with Anchor and Rust for staking SPL tokens, reward distribution, and asset management.
- [Repo](https://github.com/casaislabs/StakeFlow) · [Live](https://stakeflow.casaislabs.com)

### MessageVault
EIP-4337 smart-account dApp for on-chain message storage with The Graph indexing.
- [Repo](https://github.com/casaislabs/MessageVault) · [Live](https://messagevault.casaislabs.com)

### SolSplit
Solana multi-send tool using v0 transactions and Address Lookup Tables (ALTs) for large recipient batches.
- [Repo](https://github.com/casaislabs/SolSplit) · [Live](https://solsplit.casaislabs.com)

### HoneyGuard
Deception and telemetry lab with honeypot services, attacker interaction logging, and analytics.
- [Repo](https://github.com/casaislabs/HoneyGuard)
